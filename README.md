# Shopping Cart

## Requirements


- JDK 1.8_0_211
- Maven 3.6

## Install


    docker-compose build
    docker-compose up -d


## Start

     docker-compose exec app

     mvn exec:java -Dexec.mainClass="com.shoppingcart.Main"
